CSS stands for Cascading Style Sheet which is used to design the webpageas.

There are three types of CSS

        i. Inline/Embeded CSS: CSS which is written inside the HTML element
        ii. Internal CSS: which is written inside the same web page in head section.
        iii. External CSS: which is written in external file making .css file and connect them with HTML pages. 
        It is a very good practice.
